## Run Locally

Clone the project

```bash
  git clone https://github.com/igtannu/Auotmation_Assentment_Amber
```

Go to the project directory

```bash
  cd Auotmation_Assentment_Amber
```

To Run Maven

```bash
  double Click on Run.bat file
```
